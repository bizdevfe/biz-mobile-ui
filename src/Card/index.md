---
卡片 Card
---
仅提供卡片容器, 卡片内容由开发者自定义。

## API

| 属性      | 类型    | 默认值       | 描述         |
|----------|---------|------------|--------------|
|prefixCls |string   |biz-card  | 自定义组件主题类名前缀|
|className | string  |-           |组件跟节点添加类名|
|style|Object|{}|自定义组件跟节点的内联样式|
|full      |boolean   | false |是否占据整个父节点区域, 如是,卡片没有圆角,如否,卡片两侧各留白10px|
