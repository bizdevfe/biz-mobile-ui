---
省略号 Ellipsis
---

内容超过制定长度后显示省略号(支持多行)。

## API

| 属性      | 类型    | 默认值       | 描述         |
|----------|---------|------------|--------------|
|prefixCls |string   |biz-ellipsis  | 自定义组件主题类名前缀|
|className | string  |-           |组件跟节点添加类名|
|style|Object|{}|自定义组件跟节点的内联样式|
|line|int|1|文本行数|
|text|string||文本内容|
|width|string|-|宽度|
